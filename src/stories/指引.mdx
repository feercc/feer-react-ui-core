# 介绍

**feer-react-ui-core** 是一个强大的组件库，它包含了 react 组件，常用 hooks，以及一些常用的工具函数。它的目标是提供一套高质量的组件，让开发者可以快速的搭建出一个高质量的 react 应用。当然还有一些常用的 UI 样式，能够让你快速复用，并简单介绍了其中的原理，欢迎给我们意见和建议


# 安装

```bash
npm install feer-react-ui-core --save
```

# 使用

需要全局引入样式

```jsx
// 引入
import 'feer-react-ui-core/@styles';
import { Button } from 'feer-react-ui-core';

// 使用
<Button>按钮</Button>

```
